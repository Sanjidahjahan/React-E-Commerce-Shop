**React-E-Commerce-Shop**
- A fully functional e-commerce interface built in React, transformed from a static HTML/CSS design into a modular, component-driven shopping experience.

**Live Demo**
- https://sanjidahjahan.github.io/React-E-Commerce-Shop/

**Features**
- Browse a grid of products on the home page
- View detailed product information on individual product pages
- Add items to the cart with quantity controls
- Real-time cart updates with running totals
- Remove items individually or adjust quantities directly from the cart
- Seamless page transitions using React Router
- Responsive layout across screen sizes

**Tech Stack**
- React
- React Router v6
- HTML / CSS
- JavaScript

**What I Learned**
- This project taught me how to bridge design and engineering, breaking a static mockup into reusable, stateful components while keeping the interface visually consistent. Key challenges included managing shared cart state across pages, implementing dynamic routing, and debugging layout inconsistencies across components. It strengthened my understanding of React's component model, props, state, and the importance of clean architecture for scalability.
